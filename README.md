**Zenith Engine repository.**  

**Latest Version:** **0.1.0-alpha**  

## **About Zenith**  
Zenith is an open-source game engine built from the ground up in C++, designed to be both powerful and flexible for modern 3D game development. The goal is to provide a robust rendering pipeline, efficient resource management, and a modular architecture that supports a wide range of game development needs.  

## **Vision & Goals**  
The plan for Zenith is to develop a high-performance, feature-rich 3D game engine that:  
- **Offers a modern rendering pipeline** (supporting Deferred Rendering).  
- **Utilizes Vulkan** for optimized graphics performance.  
- **Provides an extensible architecture** for game developers and engine modifications.  
- **Supports a component-based entity system** for game object management.  
- **Enables efficient asset management** with an optimized resource loader.
